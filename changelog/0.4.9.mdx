# v0.4.9

**Released:** August 27, 2025
**Author:** [@aaron-siegel](https://github.com/aaron-siegel)
**View on GitHub:** [v0.4.9](https://github.com/pixeltable/pixeltable/releases/tag/v0.4.9)

## What's Changed
* WhisperX Speaker Diarization by @aaron-siegel in https://github.com/pixeltable/pixeltable/pull/770
* Basic support for concurrent pixeltable metadata creation/upgrade by @amithadke in https://github.com/pixeltable/pixeltable/pull/769
* Support for pydantic models in Table.insert() by @mkornacker in https://github.com/pixeltable/pixeltable/pull/760
* Add comments for concurrent pixeltable initialization changes by @amithadke in https://github.com/pixeltable/pixeltable/pull/772
* Disable notebook tests that are failing in CI for unknown reasons by @aaron-siegel in https://github.com/pixeltable/pixeltable/pull/777
* Publish the existing mypy plugin under `pixeltable.mypy` module to make it accessible for external use. by @amithadke in https://github.com/pixeltable/pixeltable/pull/776
* Remove `ext` package and fold contents into `functions` by @aaron-siegel in https://github.com/pixeltable/pixeltable/pull/775

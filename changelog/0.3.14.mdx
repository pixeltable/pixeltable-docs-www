## What's Changed
* PXT-480: Add DATE type to pixeltable supported types. by @jpeterson-pxt in https://github.com/pixeltable/pixeltable/pull/599
* PXT-560: Combine ALTER TABLE operations when adding or dropping columns. by @jpeterson-pxt in https://github.com/pixeltable/pixeltable/pull/608
* Remove support for Python 3.9 by @aaron-siegel in https://github.com/pixeltable/pixeltable/pull/596
* PXT-514 Add distinct to table, dataframe and columnref by @amithadke in https://github.com/pixeltable/pixeltable/pull/598
* update pixelagent links by @jacobweiss2305 in https://github.com/pixeltable/pixeltable/pull/615
* Add Bedrock integration notebook with basic chat and tool-based agent by @jacobweiss2305 in https://github.com/pixeltable/pixeltable/pull/600
* Updates to UDF API docs [rough-edges] by @aaron-siegel in https://github.com/pixeltable/pixeltable/pull/616
* Docs/building with llms and chg log v3.13 by @jacobweiss2305 in https://github.com/pixeltable/pixeltable/pull/610
* Properly resolve `ColumnRef`s whose reference table differs from the column definition table by @aaron-siegel in https://github.com/pixeltable/pixeltable/pull/582
* change url to include keyword llms by @jacobweiss2305 in https://github.com/pixeltable/pixeltable/pull/619
* Add make_video examples to documentation by @jacobweiss2305 in https://github.com/pixeltable/pixeltable/pull/618
* Logic for reconstituting replica data from a published bundle; new `pxt.create_replica()` method by @aaron-siegel in https://github.com/pixeltable/pixeltable/pull/581
* Add configuration page with system settings and API configuration by @jacobweiss2305 in https://github.com/pixeltable/pixeltable/pull/622
* Switch from deprecated `google-generativeai` package to newer `google-genai` by @aaron-siegel in https://github.com/pixeltable/pixeltable/pull/627
* PXT-573 fixing planner bugs related to aggregate ordering by @mkornacker in https://github.com/pixeltable/pixeltable/pull/629
* Update object-detection-in-videos notebook by @aaron-siegel in https://github.com/pixeltable/pixeltable/pull/630

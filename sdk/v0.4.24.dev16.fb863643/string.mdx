# `module` pixeltable.functions.string

<a href="https://github.com/pixeltable/pixeltable/blob/main/pixeltable/functions/string.py#L0" target="_blank">View source on GitHub</a>


Pixeltable [UDFs](https://pixeltable.readme.io/docs/user-defined-functions-udfs) for `StringType`. It closely follows the Pandas `pandas.Series.str` API.

Example:

``` python
import pixeltable as pxt

t = pxt.get_table(...)
t.select(t.str_col.capitalize()).collect()
```

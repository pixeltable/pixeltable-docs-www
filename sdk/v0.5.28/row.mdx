# <span style={{ 'color': 'gray' }}>class</span>&nbsp; pixeltable.Row


A dict-like wrapper over a single result row.

Supports key access (`row['col']`), membership (`'col' in row`),
iteration over keys, and the standard `get`, `keys`, `values`,
and `items` methods.

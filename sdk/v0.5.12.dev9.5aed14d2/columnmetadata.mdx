# <span style={{ 'color': 'gray' }}>class</span>&nbsp; pixeltable.ColumnMetadata


Metadata for a column of a Pixeltable table.

## <span style={{ 'color': 'gray' }}>attr</span>&nbsp; computed_with

```
computed_with: str | None
```

Expression used to compute this column; `None` if this is not a computed column.

## <span style={{ 'color': 'gray' }}>attr</span>&nbsp; defined_in

```
defined_in: str | None
```

Name of the table where this column was originally defined.

If the current table is a view, then `defined_in` may differ from the current table name.

## <span style={{ 'color': 'gray' }}>attr</span>&nbsp; is_primary_key

```
is_primary_key: bool
```

`True` if this column is part of the table's primary key.

## <span style={{ 'color': 'gray' }}>attr</span>&nbsp; is_stored

```
is_stored: bool
```

`True` if this is a stored column; `False` if it is dynamically computed.

## <span style={{ 'color': 'gray' }}>attr</span>&nbsp; media_validation

```
media_validation: Literal['on_read', 'on_write'] | None
```

The media validation policy for this column.

## <span style={{ 'color': 'gray' }}>attr</span>&nbsp; name

```
name: str
```

The name of the column.

## <span style={{ 'color': 'gray' }}>attr</span>&nbsp; type_

```
type_: str
```

The type specifier of the column.

## <span style={{ 'color': 'gray' }}>attr</span>&nbsp; version_added

```
version_added: int
```

The table version when this column was added.

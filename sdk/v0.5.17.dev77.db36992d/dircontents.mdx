# <span style={{ 'color': 'gray' }}>class</span>&nbsp; pixeltable.DirContents

## <span style={{ 'color': 'gray' }}>attr</span>&nbsp; dirs

```
dirs: list[str]
```

List of directory paths contained in this directory.

## <span style={{ 'color': 'gray' }}>attr</span>&nbsp; tables

```
tables: list[str]
```

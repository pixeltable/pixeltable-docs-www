# module `pixeltable.functions.json`

<a href="https://github.com/pixeltable/pixeltable/blob/main/pixeltable/functions/json.py#L0" target="_blank">View source on GitHub</a>


Pixeltable [UDFs](https://pixeltable.readme.io/docs/user-defined-functions-udfs) for `JsonType`.

Example:
```python
import pixeltable as pxt
import pixeltable.functions as pxtf

t = pxt.get_table(...)
t.select(pxtf.json.make_list(t.json_col)).collect()
```


## udf `make_list()`

```python
make_list(obj: Json) -> Json
```

Collects arguments into a list.
